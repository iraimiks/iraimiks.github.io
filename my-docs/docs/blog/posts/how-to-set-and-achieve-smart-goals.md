# How to set and achieve SMART goals

It might not be about SMART goals specifically, but similar techniques can be used. These methods provide a framework for achieving goals, but I think you need to adapt your approach to fit your own style—take the best elements from each technique and apply them step by step.
Start with a basic action that aligns with the technique you’re using, and iterate until you reach the end. Each technique needs to be understood and learned through practice, and it can sometimes be frustrating to figure out where to begin. My best advice is to start with the first step that interests you and enjoy the process until you reach your goal.
