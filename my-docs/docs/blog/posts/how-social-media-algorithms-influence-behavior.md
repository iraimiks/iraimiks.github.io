# How social media algorithms influence behavior

It was fun in the early days of social media when it was still new, but now I try my best not to use it. I use some websites that don’t have algorithms implemented—I enjoy my limited time consuming that content, and that’s all.
Social media is a rabbit hole where it seems like you’re trying to find something you saw at the beginning, but you don’t know where it will take you. Overall, it’s a bad ride. I prefer the "real-life algorithm," where you can meet random people and start talking to them.
