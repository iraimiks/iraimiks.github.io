# The rise of remote work tools

I think it’s fantastic when this option becomes available. As I understand it, this shift began when COVID-19 spread around the world. It’s so interesting how humans can adapt their daily work and habits.
I believe this evolution and flexibility are skills that must be developed and improved. Mostly, I use communication tools like video or messaging software, but in general, you can send all kinds of data to each other and solve real-world problems by delivering value remotely.
I’m still learning and doing it, and I hope I will succeed.
