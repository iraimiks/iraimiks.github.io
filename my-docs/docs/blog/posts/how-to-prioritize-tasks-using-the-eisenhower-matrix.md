# How to prioritize tasks using the Eisenhower Matrix

From what I understand, the key point of using the Eisenhower Matrix is to prioritize tasks. Urgent tasks are blockers that prevent you from moving forward. I think anything that could harm you in the future needs to be addressed by predicting how bad it could be if you don’t handle it in a short time and instead leave it for later.
Everything we do reflects on our future selves. If something can be done right now, you should just do it—even if getting that first impulse to act is hard. It’s all an internal dialogue with yourself about what’s truly good for you.
