# The Pomodoro Technique: Does it work?

It’s a great technique to help you focus more on what must be done. In some cases, it can really help you get the work done, or if you need to learn something, it can create good discipline and dedication.
What seems to work best is turning processes into routines that become a natural part of who you are. For example, if you want to improve your writing, just start by writing. Try to find techniques on how to write, create a simple plan, and execute it. I can already feel the results.
