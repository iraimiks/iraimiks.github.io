# The importance of coding for non-developers

With coding, you can exercise your brain in a systematic way, understanding where things begin and end. Coding helps you plan actions in your mind: you prepare what you want to do, and the next step is to write the code with your fingers on the keyboard.
Sometimes it’s hard, but other times, the code flows effortlessly—that’s a great feeling. For non-developers, if they work in a field where they need to automate tasks with a computer, understanding how things work with computers will somehow benefit them in the long run.
